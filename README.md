# Carnet d'adresses (Python - Console)

## Description
Ce projet est une application en ligne de commande permettant de gérer un carnet d'adresses.  
L'utilisateur peut ajouter des contacts et afficher la liste des contacts enregistrés.

## Objectif
J'ai réalisé ce projet pour m'entraîner à :
- manipuler des structures de données (dictionnaires)
- gérer les entrées utilisateur
- créer une boucle interactive en Python

## Fonctionnalités
- Ajouter un contact (nom, téléphone, email, ville)
- Stocker les contacts dans un dictionnaire
- Afficher tous les contacts enregistrés
- Quitter le programme à tout moment

## Fonctionnement
1. L'utilisateur entre les informations d'un contact
2. Le contact est enregistré dans un dictionnaire
3. L'utilisateur peut :
   - appuyer sur Entrée pour ajouter un autre contact
   - taper `a` pour afficher les contacts
   - taper `q` pour quitter

## Technologies utilisées
- Python

1. Installer Python
